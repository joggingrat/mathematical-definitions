# mathematical-definitions
I work in ZFC with the usual set theoretical notations. These are:
1. $`\emptyset`$,$`0`$
2. $`a\cap b`$
3. $`a\cup b`$ 
4. $`a-b`$
5. $`\bigcup a`$
6. $`\bigcap a`$
7. $`\{\mathbf x|\varphi\}`$
8. $`\{\tau|\varphi\}`$
9. $`\{a,b\}`$
10. $`\{a,b,c\}`$
11. $`\cdots`$
12. $`\langle a,b\rangle`$
13. $`a\times b`$
14. $`\mathcal{D}a`$
15. $`\mathcal{R}a`$
16. $`a|b`$
17. $`a"b`$
18. $`a^b`$
19. $`a:b\rightarrow c`$
20. $`a(b)`$,$`a_b`$,$`a\ b`$
21. $`N`$
22. $`|a|`$
23. $`\langle a\rangle`$
24. $`\langle a,b\rangle`$
25. $`\cdots`$
26. $`\aleph_a`$
## abbreviations
1. $`\forall n(\varphi)`$ -> $`\forall a(a\in N\rightarrow \varphi)`$
2. $`\exists n(\varphi)`$ -> $`\exists a(a\in N\wedge\varphi)`$
3. $`\varphi,\psi`$ -> $`\varphi\wedge\psi`$
4. $`a<b`$ -> $`\langle a,b\rangle\in <`$
5. $`a+b`$ -> $`+(\langle a,b\rangle)`$
6. $`a(b)(c)(d)`$ -> $`((a(b))(c))(d)`$
7. $`a\ b\ c\ d`$ -> $`((a\ b)\ c)\ d`$
